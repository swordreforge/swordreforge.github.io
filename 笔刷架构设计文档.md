# 笔刷架构设计文档

## 1. 概述

本文档描述了基于 photon-rs 的 WebAssembly 图像处理器中笔刷涂鸦功能的架构设计。该架构旨在为用户提供流畅、自然的绘画体验，支持多种笔刷风格和效果。

### 1.1 设计目标

- **自然手绘效果**：实现平滑、自然的笔触效果，支持压感和速度响应
- **高性能渲染**：在 WebAssembly 环境中提供流畅的实时绘制体验
- **可扩展性**：支持多种笔刷类型和效果，便于后续扩展
- **多格式支持**：支持 PNG、JPEG、WEBP 等多种图像格式的导入导出
- **跨平台兼容**：在浏览器和原生环境中保持一致的 API

## 2. 技术栈选型

### 2.1 核心库

| 库名 | 版本 | 用途 |
|------|------|------|
| **photon-rs** | 最新 | 图像处理核心，提供 PhotonImage 数据结构 |
| **tiny-skia** | 最新 | 2D 矢量渲染引擎，负责笔触绘制 |
| **freedraw** | 最新 | 手绘笔迹生成，提供自然的线条平滑算法 |
| **wasm-bindgen** | 最新 | WebAssembly 绑定，提供 JavaScript 接口 |

### 2.2 选型理由

#### photon-rs
- 提供统一的 `PhotonImage` 数据结构，包含原始像素数据（RGBA 格式）
- 支持多种图像格式的编解码（PNG、JPEG、WEBP、BMP、ICO、TIFF）
- 已有完整的图像处理功能（滤镜、变换、通道操作等）
- 原生支持 WebAssembly 环境

#### tiny-skia
- 纯 CPU 2D 渲染库，无需 GPU 依赖
- 支持路径填充、描边、渐变、混合模式、抗锯齿
- 二进制体积小，运行速度快
- 完美的与 `PhotonImage` 数据结构集成

#### freedraw
- Rust 移植自 `perfect-freehand` 库
- 专门用于生成平滑手绘线条
- 支持压感、锥度、平滑度等参数
- 可输出 SVG 路径，便于转换为 `tiny-skia::Path`

### 3.2 模块划分

#### 3.2.1 交互层（JavaScript）
- **职责**：处理用户输入事件（鼠标/触控）、管理画布状态、配置笔刷参数
- **主要组件**：
  - `CanvasManager`：画布管理器
  - `InputHandler`：输入事件处理器
  - `BrushConfig`：笔刷配置

#### 3.2.2 笔迹处理层（Rust - WASM）
- **职责**：将原始输入点转换为平滑的矢量路径
- **主要组件**：
  - `StrokeGenerator`：笔划生成器
  - `PathBuilder`：路径构建器
  - `freedraw` 集成

#### 3.2.3 渲染层（Rust - WASM）
- **职责**：将矢量路径渲染到像素缓冲区
- **主要组件**：
  - `Renderer`：渲染器
  - `BrushPainter`：笔刷绘制器
  - `tiny-skia` 集成

#### 3.2.4 数据层（Rust - WASM）
- **职责**：管理图像数据，提供格式转换
- **主要组件**：
  - `DataConverter`：数据转换器
  - `PhotonImage`：photon-rs 核心数据结构
  - `Pixmap`：tiny-skia 渲染目标

```rust
/// PhotonImage 与 Pixmap 之间的转换
pub struct DataConverter;

impl DataConverter {
    /// 从 PhotonImage 创建 Pixmap
    pub fn photon_to_pixmap(photon: &PhotonImage) -> Option<Pixmap> {
        Pixmap::from_vec(
            photon.raw_pixels().clone(),
            photon.get_width(),
            photon.get_height()
        )
    }

    /// 将 Pixmap 数据写回 PhotonImage
    pub fn pixmap_to_photon(pixmap: &Pixmap, photon: &mut PhotonImage) {
        photon.raw_pixels_mut()
            .copy_from_slice(pixmap.data());
    }
}
```

### 5.2 渲染模块

```rust
/// 笔刷渲染器
pub struct BrushRenderer;

impl BrushRenderer {
    /// 渲染一笔到图像
    pub fn render_stroke(
        image: &mut PhotonImage,
        stroke: &BrushStroke,
    ) -> Result<(), String> {
        // 1. 将 PhotonImage 转换为 Pixmap
        let mut pixmap = DataConverter::photon_to_pixmap(image)
            .ok_or("Failed to create pixmap")?;

        // 2. 获取或生成路径
        let path = stroke.path.as_ref()
            .ok_or("Path not generated")?;

        // 3. 配置画笔样式
        let paint = self.create_paint(&stroke.config);
        let stroke_style = self.create_stroke_style(&stroke.config);

        // 4. 渲染路径
        pixmap.stroke_path(
            path,
            &paint,
            &stroke_style,
            tiny_skia::Transform::identity(),
            None,
        );

        // 5. 将结果写回 PhotonImage
        DataConverter::pixmap_to_photon(&pixmap, image);

        Ok(())
    }

    /// 创建画笔
    fn create_paint(config: &BrushConfig) -> tiny_skia::Paint {
        let mut paint = tiny_skia::Paint::default();
        
        // 设置颜色
        paint.set_color_rgba8(
            config.color.0,
            config.color.1,
            config.color.2,
            config.color.3,
        );
        
        // 设置抗锯齿
        paint.anti_alias = true;
        
        // 设置混合模式
        paint.blend_mode = self.map_blend_mode(config.blend_mode);
        
        paint
    }

    /// 创建描边样式
    fn create_stroke_style(config: &BrushConfig) -> tiny_skia::Stroke {
        tiny_skia::Stroke {
            width: config.base_width,
            line_cap: tiny_skia::LineCap::Round,
            line_join: tiny_skia::LineJoin::Round,
            ..Default::default()
        }
    }

    /// 映射混合模式
    fn map_blend_mode(mode: BlendMode) -> tiny_skia::BlendMode {
        match mode {
            BlendMode::Normal => tiny_skia::BlendMode::Over,
            BlendMode::Multiply => tiny_skia::BlendMode::Multiply,
            BlendMode::Screen => tiny_skia::BlendMode::Screen,
            BlendMode::Overlay => tiny_skia::BlendMode::Over,
            BlendMode::SoftLight => tiny_skia::BlendMode::SoftLight,
            BlendMode::HardLight => tiny_skia::BlendMode::HardLight,
            BlendMode::Difference => tiny_skia::BlendMode::Difference,
            BlendMode::Exclusion => tiny_skia::BlendMode::Exclusion,
            BlendMode::Lighten => tiny_skia::BlendMode::Lighten,
            BlendMode::Darken => tiny_skia::BlendMode::Darken,
        }
    }
}
```

```typescript
        // 触控事件
        this.canvas.addEventListener('touchstart', (e) => this.onTouchStart(e));
        this.canvas.addEventListener('touchmove', (e) => this.onTouchMove(e));
        this.canvas.addEventListener('touchend', () => this.onTouchEnd());
    }

    private getCanvasCoordinates(e: MouseEvent | TouchEvent): Point {
        const rect = this.canvas.getBoundingClientRect();
        const scaleX = this.canvas.width / rect.width;
        const scaleY = this.canvas.height / rect.height;

        if (e instanceof MouseEvent) {
            return {
                x: (e.clientX - rect.left) * scaleX,
                y: (e.clientY - rect.top) * scaleY,
            };
        } else {
            const touch = e.touches[0];
            return {
                x: (touch.clientX - rect.left) * scaleX,
                y: (touch.clientY - rect.top) * scaleY,
            };
        }
    }
}
```

```rust
/// 笔划历史管理
pub struct StrokeHistory {
    strokes: Vec<BrushStroke>,
    max_size: usize,
    undo_stack: Vec<BrushStroke>,
    redo_stack: Vec<BrushStroke>,
}

impl StrokeHistory {
    pub fn new(max_size: usize) -> Self {
        Self {
            strokes: Vec::with_capacity(max_size),
            max_size,
            undo_stack: Vec::new(),
            redo_stack: Vec::new(),
        }
    }

    pub fn add_stroke(&mut self, stroke: BrushStroke) {
        self.undo_stack.clear();
        self.redo_stack.clear();
        self.strokes.push(stroke);
        
        // 限制历史大小
        if self.strokes.len() > self.max_size {
            self.strokes.remove(0);
        }
    }

    pub fn undo(&mut self) -> Option<BrushStroke> {
        if let Some(stroke) = self.strokes.pop() {
            self.undo_stack.push(stroke);
            Some(stroke)
        } else {
            None
        }
    }
}
```

1. **自然手绘效果**：通过 freedraw 的平滑算法实现自然笔触
2. **多格式支持**：借助 photon-rs 的格式支持能力
3. **高性能渲染**：使用 tiny-skia 的纯 CPU 渲染
4. **可扩展架构**：插件式笔刷系统，便于扩展
5. **完整的数据流**：从输入到输出的完整数据处理链路

该架构为后续开发提供了坚实的基础，可以根据实际需求逐步实现和优化各个模块。
