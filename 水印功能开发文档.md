# 水印功能开发文档

## 功能概述

实现图片水印功能，支持将水印图像叠加到主图像上，并提供多种混合模式选择。

## 原始函数分析

### ✅1. watermark 函数

```rust
pub fn watermark(img: &mut PhotonImage, watermark: &PhotonImage, x: i64, y: i64)
```
- **功能**：简单叠加水印
- **参数**：
  - `img` - 主图像
  - `watermark` - 水印图像
  - `x` - X 坐标
  - `y` - Y 坐标
- **实现**：使用 `image::imageops::overlay` 进行简单叠加

### 2. blend 函数
```rust
pub fn blend(photon_image: &mut PhotonImage, photon_image2: &PhotonImage, blend_mode: &str)
```

**支持的混合模式**（15种）：
1. `overlay` - 叠加（默认）
2. `over` - 覆盖
3. `atop` - 在上
4. `xor` - 异或
5. `plus` - 相加
6. `multiply` - 正片叠底
7. `burn` - 烧毁
8. `difference` - 差值
9. `soft_light` - 柔光
10. `screen` - 滤色
11. `hard_light` - 强光
12. `dodge` - 颜色减淡
13. `exclusion` - 排除
14. `lighten` - 变亮
15. `darken` - 变暗

**限制**：第一张图像（水印）必须小于第二张图像（主图像）

## 实现方案

### 方案一：基础版（已实现）

**导出函数**：
- `apply_watermark(watermark_bytes, x, y)` - 简单叠加

**Rust 实现**：
```rust
pub fn apply_watermark(&mut self, watermark_bytes: &[u8], x: i64, y: i64) {
    let watermark = native::open_image_from_bytes(watermark_bytes)
        .expect("Failed to open watermark image");
    photon_rs::multiple::watermark(&mut self.image, &watermark, x, y);
}
```

**UI 设计**：
- 水印图片上传按钮
- X 坐标滑块（0 到 图像宽度）
- Y 坐标滑块（0 到 图像高度）
- 应用按钮

### 方案二：增强版（推荐）

**导出函数**：
- `apply_watermark_with_blend(watermark_bytes, x, y, blend_mode)` - 带混合模式

**Rust 实现**：
```rust
pub fn apply_watermark_with_blend(&mut self, watermark_bytes: &[u8], x: i64, y: i64, blend_mode: &str) {
    let watermark = native::open_image_from_bytes(watermark_bytes)
        .expect("Failed to open watermark image");
    
    // 调整水印大小以适应主图像
    let watermark = if watermark.get_width() > self.width || watermark.get_height() > self.height {
        // 计算缩放比例
        let scale = (self.width as f32 / watermark.get_width() as f32)
            .min(self.height as f32 / watermark.get_height() as f32) * 0.3;
        let new_width = (watermark.get_width() as f32 * scale) as u32;
        let new_height = (watermark.get_height() as f32 * scale) as u32;
        transform::resize(&watermark, new_width, new_height, transform::SamplingFilter::Lanczos3)
    } else {
        watermark
    };
    
    photon_rs::multiple::blend(&mut self.image, &watermark, blend_mode);
}
```

**UI 设计**：
- 水印图片上传区域（支持拖拽）
- 混合模式下拉框（15种模式）
- X 坐标滑块
- Y 坐标滑块
- 水印大小滑块（缩放比例）
- 预览功能
- 应用/移除按钮

### 方案三：高级版（可选）

**导出函数**：
- `apply_watermark_with_opacity(watermark_bytes, x, y, opacity)` - 带透明度
- `apply_text_watermark(text, x, y, font_size, opacity)` - 文字水印

**UI 设计**：
- 支持图片水印和文字水印
- 透明度滑块（0-100%）
- 水印旋转角度
- 水印阴影效果

## 注意事项

1. **水印大小限制**：blend 函数要求水印必须小于主图像
2. **坐标范围**：X/Y 坐标不能超出图像边界
3. **性能优化**：大图片水印可能影响性能，建议先缩放
4. **格式支持**：支持 JPG、PNG、WebP 等常见格式
