# 不规则形状抠图实现方案

## 概述

在 Rust 中实现任意不规则形状的抠图，核心思路是通过遮罩进行图像裁剪。这个过程包含三个主要步骤：生成形状遮罩、应用遮罩合成 RGBA 图像、边缘优化。

## 技术路径

### 依赖项

项目已依赖 `image = "0.24.8"`，建议额外添加：

```toml
[dependencies]
image = "0.24.8"
imageproc = "0.23"
tiny-skia = "0.11"
```

## 实现步骤

### 第一步：生成形状遮罩 (Mask)

创建一个与原始图像尺寸相同的灰度图像，在要保留的形状区域内填充白色（255），在要抠掉的区域填充黑色（0）。

#### 方案 A：使用 `imageproc` 绘制多边形遮罩

```rust
use image::{ImageBuffer, Luma};
use imageproc::drawing::draw_polygon;
use imageproc::point::Point;

fn create_polygon_mask(width: u32, height: u32, vertices: Vec<(i32, i32)>) -> ImageBuffer<Luma<u8>, Vec<u8>> {
    // 创建全黑遮罩画布（背景透明）
    let mut mask = ImageBuffer::from_pixel(width, height, Luma([0u8]));

    // 转换顶点格式
    let points: Vec<Point<i32>> = vertices.into_iter()
        .map(|(x, y)| Point::new(x, y))
        .collect();

    // 绘制白色填充多边形
    draw_polygon(&mut mask, &points, Luma([255u8]));

    mask
}
```

#### 方案 B：使用 `tiny-skia` 生成抗锯齿遮罩

```rust
use tiny_skia::{Pixmap, PathBuilder, Paint, Stroke, Transform};

fn create_anti_aliased_mask(width: u32, height: u32, vertices: Vec<(f32, f32)>) -> Pixmap {
    let mut pixmap = Pixmap::new(width, height).unwrap();

    // 构建路径
    let mut path_builder = PathBuilder::new();
    if let Some((first_x, first_y)) = vertices.first() {
        path_builder.move_to(*first_x, *first_y);
        for (x, y) in vertices.iter().skip(1) {
            path_builder.line_to(*x, *y);
        }
        path_builder.close();
    }

    let path = path_builder.finish().unwrap();

    // 设置白色填充
    let mut paint = Paint::default();
    paint.set_color(tiny_skia::Color::WHITE);

    // 绘制抗锯齿路径
    pixmap.fill_path(
        &path,
        &paint,
        Transform::identity(),
        None,
    );

    pixmap
}
```

### 第二步：应用遮罩合成 RGBA 图像

将原始图像与遮罩图像结合，生成带有透明背景的新图像。

```rust
use image::{ImageBuffer, Rgb, Rgba, Luma};

fn apply_mask_to_image(
    image: &ImageBuffer<Rgb<u8>, Vec<u8>>,
    mask: &ImageBuffer<Luma<u8>, Vec<u8>>,
) -> ImageBuffer<Rgba<u8>, Vec<u8>> {
    let (width, height) = image.dimensions();
    let mut output = ImageBuffer::new(width, height);

    for (x, y, output_pixel) in output.enumerate_pixels_mut() {
        let rgb = image.get_pixel(x, y);
        let alpha = mask.get_pixel(x, y).0[0];

        *output_pixel = Rgba([rgb.0[0], rgb.0[1], rgb.0[2], alpha]);
    }

    output
}
```

### 第三步：边缘优化 (Refine Edges)

对遮罩边缘进行抗锯齿或羽化处理，使抠图效果更自然。

#### 抗锯齿实现

对于圆形等几何形状，通过计算像素到边界的距离，赋予中间灰度值：

```rust
fn create_circular_mask_anti_aliased(
    width: u32,
    height: u32,
    center: (f32, f32),
    radius: f32,
) -> ImageBuffer<Luma<u8>, Vec<u8>> {
    let mut mask = ImageBuffer::from_pixel(width, height, Luma([0u8]));
    let feather_radius = 2.0; // 羽化半径

    for (x, y, pixel) in mask.enumerate_pixels_mut() {
        let dx = x as f32 - center.0;
        let dy = y as f32 - center.1;
        let distance = (dx * dx + dy * dy).sqrt();

        let alpha = if distance <= radius - feather_radius {
            255.0
        } else if distance >= radius + feather_radius {
            0.0
        } else {
            // 线性插值
            255.0 * (1.0 - (distance - (radius - feather_radius)) / (2.0 * feather_radius))
        };

        *pixel = Luma([alpha as u8]);
    }

    mask
}
```

#### 使用形态学操作平滑边缘

```rust
// 需要添加 fast_morphology 依赖
use fast_morphology::{erode, dilate};

fn smooth_mask_edges(mask: &ImageBuffer<Luma<u8>, Vec<u8>>) -> ImageBuffer<Luma<u8>, Vec<u8>> {
    // 先轻微腐蚀，再轻微膨胀（闭运算）
    let eroded = erode(mask, 1);
    dilate(&eroded, 1)
}
```

## 完整示例：五角星抠图

```rust
use image::{ImageBuffer, Luma, Rgba, Rgb, open};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 1. 加载原始图像
    let img = open("input.jpg")?.to_rgb8();
    let (width, height) = img.dimensions();

    // 2. 生成五角星遮罩
    let vertices = generate_star_vertices(width, height, 5);
    let mask = create_polygon_mask(width, height, vertices);

    // 3. 应用遮罩
    let result = apply_mask_to_image(&img, &mask);

    // 4. 保存结果
    result.save("output.png")?;

    Ok(())
}

fn generate_star_vertices(width: u32, height: u32, points: u32) -> Vec<(i32, i32)> {
    let center_x = width as f32 / 2.0;
    let center_y = height as f32 / 2.0;
    let outer_radius = width.min(height) as f32 * 0.4;
    let inner_radius = outer_radius * 0.4;

    let mut vertices = Vec::new();
    let step = std::f32::consts::PI / points as f32;

    for i in 0..(points * 2) {
        let angle = i as f32 * step - std::f32::consts::PI / 2.0;
        let radius = if i % 2 == 0 { outer_radius } else { inner_radius };

        let x = (center_x + angle.cos() * radius) as i32;
        let y = (center_y + angle.sin() * radius) as i32;
        vertices.push((x, y));
    }

    vertices
}
```

## 技术方案对比

| 步骤 | 核心任务 | 推荐库 | 关键思路 |
|------|----------|--------|----------|
| 1. 生成遮罩 | 创建不规则形状的黑白图像 | `image` + `imageproc` / `tiny-skia` | 绘制多边形或路径，白色（255）为保留区 |
| 2. 合成图像 | 将遮罩的Alpha通道应用到原图 | `image` | 遍历像素，组合 RGB 与灰度值 |
| 3. 边缘优化 | 消除锯齿，实现平滑过渡 | 手动实现或 `tiny-skia` | 生成非二值遮罩，边缘像素具有中间灰度 |

## 集成到 photon-wasm

在 `photon-wasm` 项目中，可以添加以下 WASM 接口：

```rust
#[wasm_bindgen]
pub struct ImageProcessor {
    // 现有字段...
}

#[wasm_bindgen]
impl ImageProcessor {
    // 现有方法...

    /// 应用多边形遮罩
    pub fn apply_polygon_mask(&mut self, vertices: Vec<f32>) {
        let width = self.width;
        let height = self.height;
        let mask = create_polygon_mask(width, height, convert_vertices(vertices));
        self.apply_mask(&mask);
    }

    /// 应用圆形遮罩
    pub fn apply_circular_mask(&mut self, center_x: f32, center_y: f32, radius: f32) {
        let mask = create_circular_mask_anti_aliased(
            self.width,
            self.height,
            (center_x, center_y),
            radius,
        );
        self.apply_mask(&mask);
    }
}
```

## 注意事项

1. **性能考虑**：遍历像素操作在 WASM 中可能较慢，考虑使用 SIMD 优化
2. **内存管理**：处理大图像时注意内存使用
3. **边界检查**：确保遮罩与原图尺寸一致
4. **Alpha 通道**：确保输出格式支持透明度（PNG）
