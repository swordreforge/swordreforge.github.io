# 字体支持开发文档

## 当前状态

### 已有字体
项目包含以下字体文件，位于 `photon-wasm/photon-master/crate/fonts/`：
- **Roboto-Regular.ttf** - 常规字体（当前唯一使用的字体）
- **Roboto-Black.ttf** - 粗体字体（已包含但未使用）
- **Apache License.txt** - 字体许可证文件

### 当前实现
所有文本绘制函数都硬编码使用 `Roboto-Regular.ttf`：
```rust
let font = Vec::from(include_bytes!("../fonts/Roboto-Regular.ttf") as &[u8]);
```

## 添加新字体的步骤

### 1. 准备字体文件

将新的字体文件（.ttf 格式）添加到项目目录：
```
photon-wasm/photon-master/crate/fonts/
```

**注意事项：**
- 字体文件必须支持 Apache 2.0 许可证
- 字体文件大小会影响 WASM 模块大小
- 建议使用 Web 字体优化工具减小文件大小

### 2. 修改 photon-rs 的 text.rs

在 `photon-wasm/photon-master/crate/src/text.rs` 中：

#### 方案 A：添加新函数（推荐）
为每个字体创建单独的函数：
```rust
pub fn draw_text_with_font_regular(
    photon_img: &mut PhotonImage,
    text: &str,
    x: i32,
    y: i32,
    font_size: f32,
    r: u8,
    g: u8,
    b: u8,
) {
    let font = Vec::from(include_bytes!("../fonts/Roboto-Regular.ttf") as &[u8]);
    // ... 绘制逻辑
}

pub fn draw_text_with_font_black(
    photon_img: &mut PhotonImage,
    text: &str,
    x: i32,
    y: i32,
    font_size: f32,
    r: u8,
    g: u8,
    b: u8,
) {
    let font = Vec::from(include_bytes!("../fonts/Roboto-Black.ttf") as &[u8]);
    // ... 绘制逻辑
}
```

#### 方案 B：添加字体枚举参数
创建字体枚举和通用函数：
```rust
#[cfg_attr(feature = "enable_wasm", wasm_bindgen)]
pub enum FontType {
    Regular = 0,
    Black = 1,
    // 添加更多字体...
}

pub fn draw_text_with_font_type(
    photon_img: &mut PhotonImage,
    text: &str,
    x: i32,
    y: i32,
    font_size: f32,
    r: u8,
    g: u8,
    b: u8,
    font_type: FontType,
) {
    let font_bytes = match font_type {
        FontType::Regular => include_bytes!("../fonts/Roboto-Regular.ttf"),
        FontType::Black => include_bytes!("../fonts/Roboto-Black.ttf"),
    };
    let font = Vec::from(font_bytes as &[u8]);
    // ... 绘制逻辑
}
```

### 3. 在 photon-wasm 添加包装方法

在 `photon-wasm/src/lib.rs` 中为 `ImageProcessor` 添加相应方法：

```rust
// 方案 A：独立函数
pub fn draw_text_with_regular(&mut self, text: &str, x: i32, y: i32, font_size: f32, r: u8, g: u8, b: u8) {
    text::draw_text_with_font_regular(&mut self.image, text, x, y, font_size, r, g, b);
}

pub fn draw_text_with_black(&mut self, text: &str, x: i32, y: i32, font_size: f32, r: u8, g: u8, b: u8) {
    text::draw_text_with_font_black(&mut self.image, text, x, y, font_size, r, g, b);
}

// 方案 B：通用函数
pub fn draw_text_with_font(&mut self, text: &str, x: i32, y: i32, font_size: f32, r: u8, g: u8, b: u8, font_type: u8) {
    let font_type = match font_type {
        0 => text::FontType::Regular,
        1 => text::FontType::Black,
        _ => text::FontType::Regular,
    };
    text::draw_text_with_font_type(&mut self.image, text, x, y, font_size, r, g, b, font_type);
}
```

### 7. 更新 JavaScript 逻辑

在 `applyText()` 函数中添加字体选择：

```javascript

if (withShadow) {
    // 根据字体类型调用不同的方法

        processor.draw_text_with_shadow_and_color(text, drawX, drawY, fontSize, r, g, b);

```

## 推荐字体列表

以下是一些开源的、支持 Apache 2.0 许可证的字体，适合用于图像编辑：

### 中文支持
- **Noto Sans CJK** - Google 开源字体，支持中日韩
- **Source Han Sans** - Adobe 开源字体
- **ZCOOL QingKe HuangYou** - 青科黑体

### 英文支持
- **Roboto** (已包含) - Google 开源字体
- **Open Sans** - 开源无衬线字体
- **Lato** - 现代无衬线字体
- **Montserrat** - 几何无衬线字体

### 字体风格
- **Serif** - 衬线字体
- **Sans-Serif** - 无衬线字体
- **Monospace** - 等宽字体
- **Handwriting** - 手写风格

## 注意事项

### WASM 模块大小
- 每个 TTF 字体文件会增加 WASM 模块大小
- 建议使用子集化工具只包含需要的字符
- 考虑使用 WOFF2 格式（需要转换支持）

### 性能考虑
- 字体加载时间会影响首次绘制
- 大字体文件可能影响编译时间
- 建议限制字体数量（3-5 个为佳）

### 许可证合规
- 确保所有字体文件都有合适的开源许可证
- 在文档中注明字体来源和许可证
- 避免使用需要商业授权的字体
