# 动态字体导入功能对接完成说明

## 概述

已完成魔改后端 photon-rs 动态字体导入功能与前端代码的对接。

## 实现的功能

### 1. WASM 后端功能

在 `photon-wasm/src/lib.rs` 中添加了新的文本绘制方法:

```rust
pub fn draw_text_with_font_name(
    &mut self,
    text: &str,
    x: i32,
    y: i32,
    font_size: f32,
    font_name: &str,
    has_shadow: Option<bool>,
    color_r: Option<u8>,
    color_g: Option<u8>,
    color_b: Option<u8>,
)
```

此方法支持直接使用已注册的字体名称绘制文本。

同时修改了旧的 `draw_text` 和 `draw_text_with_color_and_font` 方法,将字体类型枚举映射到内置字体名称,保持向后兼容:

- `font_type = 0` → "丁卯点阵体"
- `font_type = 1` → "Roboto-Regular"
- `font_type = 2` → "站酷高端黑"

新增兼容方法 `draw_text_with_shadow_and_color_and_font`。

### 2. WASM 导出的字体管理函数

以下函数已在 WASM 模块中导出并可在前端使用:

- `wasm_register_font(font_name: string, font_data: Uint8Array): void` - 注册字体
- `wasm_is_font_registered(font_name: string): boolean` - 检查字体是否已注册
- `wasm_get_registered_fonts(): string[]` - 获取已注册字体列表
- `wasm_unregister_font(font_name: string): boolean` - 移除已注册字体
- `wasm_clear_fonts(): void` - 清空所有已注册字体

### 3. 前端 UI 更新

在 `index.html` 中添加了字体管理界面:

#### 字体管理区域
- 字体文件上传控件 (支持 .ttf, .otf, .woff, .woff2 格式)
- 已注册字体列表显示
- 字体注册按钮
- 清空自定义字体按钮

#### 文本绘制区域
- 更新字体选择下拉框,支持:
  - 内置字体: `builtin-0`, `builtin-1`, `builtin-2`
  - 自定义字体: `custom-{font_name}`

### 4. 前端 JavaScript 功能

实现了以下 JavaScript 函数:

- `applyUploadedFont()` - 应用上传的字体(注册到 WASM)
- `updateRegisteredFontsList()` - 更新已注册字体列表显示
- `addCustomFontToSelect(fontName)` - 添加自定义字体到下拉框
- `removeFont(fontName)` - 删除单个字体
- `clearCustomFonts()` - 清空所有自定义字体
- `loadFontFile(file)` - 加载字体文件

### 5. 文本绘制逻辑更新

修改了 `applyText()` 函数,支持动态字体名称:

```javascript
if (fontSelect.startsWith('custom-')) {
    // 使用自定义字体(动态注册的字体)
    const fontName = fontSelect.replace('custom-', '');
    processor.draw_text_with_font_name(text, drawX, drawY, fontSize, fontName, withShadow, r, g, b);
} else {
    // 使用内置字体(兼容旧版)
    const fontType = parseInt(fontSelect.replace('builtin-', ''));
    if (withShadow) {
        processor.draw_text_with_shadow_and_color_and_font(text, drawX, drawY, fontSize, r, g, b, fontType);
    } else {
        processor.draw_text_with_color_and_font(text, drawX, drawY, fontSize, r, g, b, fontType);
    }
}
```

## 使用方法

### 上传自定义字体

1. 在侧边栏找到"字体管理"区域
2. 点击"点击上传自定义字体"上传字体文件
3. 输入字体名称(自动从文件名提取)
4. 点击"注册字体"按钮

### 使用自定义字体

1. 在"文本绘制"区域选择"字体选择"
2. 从下拉框中选择已注册的自定义字体(显示为"自定义: {font_name}")
3. 输入文本内容
4. 调整字体大小、颜色等参数
5. 点击"添加文本"
6. 在画布上拖动文本框到合适位置
7. 点击"确认"应用文本

### 管理已注册字体

- 在"已注册字体"列表中查看所有已注册的自定义字体
- 点击"删除"按钮移除单个字体
- 点击"清空自定义"按钮移除所有自定义字体

## 技术细节

### 内置字体映射

WASM 后端内置了三个字体,通过字体名称访问:

- "丁卯点阵体" - 点阵风格字体,适合复古效果
- "Roboto-Regular" - Roboto 常规字体,适合现代设计
- "站酷高端黑" - 粗体中文字体,适合标题

### 字体预览限制

由于浏览器的安全限制,无法在预览时直接加载上传的自定义字体。预览使用通用字体族(如 sans-serif),实际绘制时使用 WASM 中注册的字体。

### 向后兼容性

- 旧的 `draw_text_with_color_and_font` 和 `draw_text_with_shadow_and_color_and_font` 方法仍然可用
- 内置字体通过 font_type 参数(0-2)访问
- 自定义字体通过新的 `draw_text_with_font_name` 方法访问

## 注意事项

1. 字体文件大小会影响加载时间,建议使用优化过的字体文件
2. 字体名称需要唯一,重复注册会覆盖之前的字体
3. 清空自定义字体不会影响内置字体
4. 预览和实际绘制可能存在细微差异
5. 大字体文件可能导致内存占用增加
