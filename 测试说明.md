### 问题: "图像加载失败"

**解决方案**:
1. 确保选择的是有效的图像文件

1. 增加迭代次数（至少 10 次）
2. 关闭其他浏览器标签页
3. 确保浏览器没有进入节能模式

## 💡 技术说明

### 修复的主要问题

#### 1. ImageProcessor 构造方法

**错误用法**:
```javascript
const processor = new ImageProcessor(uint8Array); // ❌
```

**正确用法**:
```javascript
const processor = ImageProcessor.new_from_bytes(uint8Array); // ✅
```

#### 2. BigInt 处理

**错误用法**:
```javascript
const size = processor.get_estimated_filesize() / 1024; // ❌
```

**正确用法**:
```javascript
const filesize = processor.get_estimated_filesize();
const size = typeof filesize === 'bigint' ? 
    Number(filesize) / 1024 : filesize / 1024; // ✅
```

#### 3. 数据验证

```javascript
const bytes = processor.get_bytes();
if (!bytes || bytes.length === 0) {
    throw new Error('无法获取图像数据');
}
```
